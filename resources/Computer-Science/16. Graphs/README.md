# ![Computer Science - Graphs](./assets/hero.png)

## Overview

In this lesson, we cover the basics of the node-based data structures known as graphs.

### Prerequisites

- Big O Notation
- Linked Lists
- Binary Trees

### Learning Objectives

By the end of this module, learners will be able to:

- Define a graph and understand the difference between undirected and directed graphs.
- Identify how graphs are used in programming.
- Implement a graph using an adjacency list.

## Content

There are two components to this module:

1. Lesson: [Graphs](./lesson/README.md)
2. Exercise: [Implementing a Graph](https://git.generalassemb.ly/modular-curriculum-all-courses/graphs-starter-code)
